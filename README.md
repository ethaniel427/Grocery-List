# Grocery List

## Overview

Grovery List is a Java console application that allows users to create and manage a grocery shoppping list. This project was created to practice java skills such as methods, array lists, while loops and for loops, and if/else 
statements.

## Features

- Add Grocery Items
- Remove Grocery Items
- Update Status of Grocery Items
- Print List of Grocery Itmes
- Exit

## Technologies Used

- Java
- BlueJ
- Scanner
- Array Lists for storing Groceries
